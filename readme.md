# Contoso     
> 一个虚拟的大学网站，包括招生、课程管理、老师管理等功能。       
> 主要目的，熟悉EF core

## Tools     
    + .Net Core 2.*
    + VS Code    
    + MySQL
    + ASP.Net Core 2
    + Entity Framework Core 2             

## Development Records     
* dotnet new mvc     
* dotnet add package Pomelo.EntityFrameworkCore.MySql --version 2.0.1
